## 通用包

### 文件相关

**[file-type](https://github.com/sindresorhus/file-type#readme)**

用于判断文件类型

```tsx
import {fileTypeFromFile} from 'file-type';

console.log(await fileTypeFromFile('Unicorn.png'));
//=> {ext: 'png', mime: 'image/png'}
```
